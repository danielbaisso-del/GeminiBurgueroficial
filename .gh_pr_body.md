O que você precisa fazer após mesclar:
1) No painel do Vercel, adicionar as variáveis de ambiente (Project > Settings > Environment Variables):
   - DATABASE_URL (obrigatório)
   - JWT_SECRET (recomendado)
   - Outras chaves (S3, Cloudinary, APIs de terceiros) se necessário

2) Para produção com Prisma em serverless, recomendamos Prisma Data Proxy para evitar exaustão de conexões; alternadamente hospede o backend num serviço com conexões persistentes.

Testes locais recomendados:
- Backend: cd backend && npm ci && npm run build
- Frontend: cd frontend && npm ci && npm run build
- Emulação Vercel: vercel dev (no root do repo) e testar endpoints /api/* (ex.: /api/health)
